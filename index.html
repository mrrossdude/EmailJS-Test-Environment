StudioVoodoo Custom Keycap Editor

Use this tool to create your own custom keycap design. Upload an image, insert symbols, or add text to make your own bespoke cap. Make sure to read these tips to get the best quality possible.

Magic Eraser Tool

If you're using images with a solid colour background like JPG or BMP, use the Magic Eraser to strip away backgrounds. Simply click the area you want to remove and hit apply.

Image Adjustment Tool

If your image is more complex or the Magic Eraser isn't working for you, try enabling the Adjustment Tool and use the sliders to fine tune your image. This tool automatically strips out white backgrounds, so you might want to click the INVERT button if your image is on a dark background.

Cropping Images

If your image has a lot of empty space or needs elements removing, use the Crop tool to cut away any excess.
